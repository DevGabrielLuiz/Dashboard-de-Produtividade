# ProdutiVibe - Dashboard de Produtividade

Este é um projeto em **Flask** que utiliza **WebSockets (SocketIO)** para gerenciar tarefas em tempo real.

## 🚀 Funcionalidades
- Adição de tarefas instantânea sem recarregar a página.
- Contador de tarefas concluídas dinâmico.
- Interface responsiva com Bootstrap 5.

## 🛠️ Como rodar o projeto
1. Crie um ambiente virtual: `python3 -m venv .venv`
2. Ative o ambiente: `source .venv/bin/activate`
3. Instale as dependências: `pip install -r requirements.txt`
4. Execute o app: `python app.py`
